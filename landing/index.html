Полный рабочий процесс

От идеи до готового YouTube-пакета.

01

Исследование

Сбор тем, заметок по источникам, поискового контекста и референсов проекта.

02

Сценарий

Генерация длинного структурированного текста со сценами, главами и проверками.

03

Голос и аватар

Подготовка озвучки, метаданных согласия для аватара и данных для провайдера.

04

Слайды

Сборка визуальных слайдов, медиа-ссылок и данных для таймлайна сцен.

05

Рендер

Передача задач в очередь рендера, контроль воркеров и сбор артефактов.

06

Публикация

Экспорт файлов, метаданных, журнала аудита и контекста для публикации на YouTube.

Интеграции

Сервисная архитектура для продакшена и локальные резервные режимы для MVP.

  • OpenAIгенерация сценариев, анализ и ассистентские процессы
  • ElevenLabsгенерация голоса и передача данных провайдеру голоса
  • Провайдеры аватаровметаданные согласия, аватары и данные для рендера
  • Pexels и Unsplashзаготовки поиска медиа для визуального ряда
  • YouTubeметаданные публикации и будущий процесс загрузки
  • PostgreSQLадаптеры учета использования, идемпотентности и долговременного состояния
  • S3-хранилищеабстракция хранения артефактов поверх локальных файлов
  • Очередь воркеровоснова внешних воркеров вместо локального пула потоков
  • OIDC и RBACуправляемая аутентификация, роли и организации
  • Stripeкаркас подписок, биллинга и учета использования

Статус готовности

Что уже есть в репозитории и что еще нужно подключить для продакшена.

Реализованный фундамент

  • FastAPI-бэкенд с проектами, конвейером, рендером и аудитом.
  • Expo-клиент для операторских рабочих процессов.
  • JSON-резервное хранилище и адаптеры для PostgreSQL.
  • Объектное хранилище, идемпотентность, обращения в поддержку и API административного аудита.
  • Структурные логи, метрики, проверки здоровья и база внешних воркеров.

Нужна продакшен-настройка

  • Управляемый PostgreSQL, миграции и политика резервного копирования и восстановления.
  • Устойчивая очередь с автоскейлингом воркеров и оповещениями.
  • Реальные ключи провайдеров поиска, голоса, аватаров, медиа и загрузки на YouTube.
  • OIDC-тенант, RBAC-политики, Stripe-продукты и вебхуки биллинга.
  • Продакшен-шаблоны рендера и юридический поток сбора согласий.

Запуск оператора

Запустите студию локально, затем подключайте продакшен-сервисы по одному.

Бэкенд

cd ai-video-studio-mvp
python -m venv .venv
.venv\Scripts\activate
pip install -r backend\requirements.txt
uvicorn backend.app.main:app --reload

Мобильное приложение

cd mobile
npm install
npx expo start
