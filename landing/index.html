Readiness status

What is already in the repository, and what still needs production wiring.

Implemented foundation

  • FastAPI backend with project, pipeline, render and audit surfaces.
  • Expo mobile client for operator workflows.
  • JSON fallback storage plus PostgreSQL-ready adapters.
  • Object storage, idempotency, support tickets and admin audit APIs.
  • Structured logging, metrics, health checks and external worker base.

Production configuration required

  • Managed PostgreSQL, migrations and backup/restore policy.
  • Durable queue deployment with worker autoscaling and alerting.
  • Real provider keys for search, voice, avatar, media and YouTube upload.
  • OIDC tenant, RBAC policy, Stripe products and billing webhooks.
  • Production render templates and legal consent collection flow.

Operator launch

Run the studio locally, then connect production services one by one.

Backend

cd ai-video-studio-mvp
python -m venv .venv
.venv\Scripts\activate
pip install -r backend\requirements.txt
uvicorn backend.app.main:app --reload

Mobile

cd mobile
npm install
npx expo start
